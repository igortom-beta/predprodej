# Lojzovy Paseky - Váš druhý dům

Vítejte na stránkách projektu **Lojzovy Paseky**!

## O projektu

Toto je web pro projekt "Váš druhý dům" běžící na **GitHub Pages** s doménou **lojzovypaseky.life**.

## Technologie

- **Hosting:** GitHub Pages
- **Doména:** lojzovypaseky.life
- **Jazyk:** HTML, CSS, JavaScript

## Jak upravit stránky

Chcete-li upravit obsah stránek:

1. Otevřete soubor `index.html`
2. Upravte obsah podle svých potřeb
3. Uložte změny a pushněte do GitHubu

Změny se automaticky projeví na webu během pár minut.
